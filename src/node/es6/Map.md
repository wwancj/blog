# Map

## 

```js
let arr=[["a",1],["b",2],["c",3]]
let ma=new Map(arr)
console.log(ma)
//会得到 一个key 和一个 value
// key => value "a"=>1 "b"=>2

```

# Map的方法
## clear（）
清空所有值
## delete(key)
删除某一项
参数：key 数据的值
返回值：value数据的值
## has(key) 

- 是否包含某一项
- 参数：key 数据的值
- 返回值：true||false 是否包含这个值

## set(key,value) 设置一个值“添加"

- 参数：key数据的值

  ​			value数据的值

- map对象的本身
